# 이 웹 서비스는 게시판을 만들기 위함입니다.

사용된 젬 목록

회원 가입에 사용됩니다.
gem 'devise', '~> 3.5', '>= 3.5.6'
부트스트랩 프레임워크를 사용하기 위해서 사용합니다.
gem 'bootstrap-sass', '~> 3.3', '>= 3.3.6'
좋아요/싫어요 기능을 넣는데 사용합니다.
gem 'acts_as_votable', '~> 0.10.0'
간단한 형태의 폼을 만드는데 유용합니다.
gem 'simple_form', '~> 3.0.2'
tinymce 리치 에디터를 사용하기 위하여 사용합니다.
gem 'tinymce-rails', '~> 4.3', '>= 4.3.3'
어썸 폰트를 사용합니다.
gem 'font-awesome-rails', '~> 4.5', '>= 4.5.0.1'
페이지네이션을 하기 위하여 사용합니다.
gem 'will_paginate-bootstrap', '~> 1.0', '>= 1.0.1'
이미지나 파일을 올릴 때 사용합니다.
gem 'paperclip', '~> 4.3', '>= 4.3.5'
올린 이미지를 아마존 웹 서비스에 올리기 위해서 사용합니다.
gem 'aws-sdk', '< 2.0'
